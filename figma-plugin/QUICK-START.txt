📦 INSTALLATION (3 STEPS)

  1. Open Figma Desktop App
     (Must use desktop, NOT browser)
  
  2. Import Plugin
     Menu (≡) → Plugins → Development → Import plugin from manifest
  
  3. Select manifest file
     Browse to: D:\buildvatana\vatana\figma-plugin\manifest.json
     Click "Open"

⚡ COMMON ISSUES & FIXES

  ❌ Plugin doesn't appear
     → Restart Figma Desktop
     → Try importing again
  
  ❌ Font errors
     → Plugin uses "Inter" font
     → Install Inter font in Figma/system
     → Or edit code.ts to use different font
  
  ❌ Nothing generates
     → Check Figma console for errors
     → Most common: Font loading issue
  
  ❌ Type errors when recompiling
     → This is NORMAL! Code still works
     → Figma provides types at runtime

🔄 MAKING CHANGES

  Edit code.ts:
    1. Make your changes
    2. Save file
    3. Run: npx tsc code.ts --target ES2017 --lib ES2017,DOM --skipLibCheck
    4. In Figma: Plugins → Development → Reload plugin
  
  Edit ui.html:
    1. Make your changes
    2. Save file
    3. In Figma: Plugins → Development → Reload plugin
